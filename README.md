# FieldTrack
This is for my dad, my favourite stakeholder, he wanted me to build him an app because he is always forgetting when to reapply bug or fungal treatment to his farm. So here is, a simple app to track product application in different zones in your land

## Goals of the project

- Provide a simple interface for tracking treatments across farm zones
- Explore modern Android development practices
- Build a practical tool for a real user

## Tech (planned / in progress)

- Kotlin
- Jetpack Compose
- MVVM architecture
- Hilt for dependency injection
- Coroutines for asynchronous work
